# Step 3: Integration Implementation Costs

## Objective

Estimate one-time and time-based implementation costs for integrating platforms.

## Instructions

Estimate per-platform implementation:

**Per-Platform Implementation Cost**:

| Activity | Hours/Cost | Rate | Total |
|---|---|---|---|
| Platform configuration | X hrs | $X/hr | $X |
| Integration setup | X hrs | $X/hr | $X |
| Data migration | X hrs | $X/hr | $X |
| Testing | X hrs | $X/hr | $X |
| Training | X hrs | $X/hr | $X |
| Vendor implementation services | Flat fee | | $X |
| Middleware/custom development | X hrs | $X/hr | $X |
| **Total implementation** | | | **$X** |

Estimate by:
- Internal team resources (full-loaded cost: $X/hour for staff engineer)
- Partner/vendor implementation services (quote from vendor)
- Custom development if needed (API integrations, middleware, custom code)
- Data migration services (if vendor-managed)

**Timeline Impact**:
- Longer timeline = more resource cost
- Compressed timeline = higher burn rate or need for more people

Show cost phasing (which costs occur in which months).

## Outputs

- Per-platform implementation cost
- Total implementation budget
- Cost phasing by month/phase

## Completion Criteria

- Implementation costs estimated for all platforms
- Costs include all major activities
- Realistic effort estimates (validated with vendors/partners)
