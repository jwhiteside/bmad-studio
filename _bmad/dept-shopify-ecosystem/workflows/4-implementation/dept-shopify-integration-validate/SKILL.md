# Shopify Integration Validation

**Entry Point**: `dept-shopify-integration-validate`

Validate that all integrations are working correctly and data is flowing between platforms as designed. Test data accuracy, sync timing, and error handling.

## What This Skill Does

- Validates data accuracy across platforms
- Validates sync timing and reliability
- Tests error handling and recovery
- Produces validation report with pass/fail status
- Identifies issues requiring remediation

## When To Use It

- Before go-live to validate all integrations
- Post-launch to confirm production readiness

## Inputs

- Integration configurations
- Data flow specifications
- Sample data for validation
- Business acceptance criteria

## Outputs

- Validation report with pass/fail per integration
- Data quality assessment
- Sync performance metrics
- Issues and remediation plan
