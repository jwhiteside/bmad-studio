# Shopify Ecosystem Go-Live

**Entry Point**: `dept-shopify-go-live`

Execute go/no-go assessment and prepare ecosystem for live launch. Run final checklists, plan cutover sequence, define rollback strategy, and produce go-live plan.

## What This Skill Does

- Gathers validation reports and integration status
- Runs go/no-go checklist per implementation phase
- Plans cutover sequence (which platforms go live when)
- Defines rollback strategy per platform
- Produces final go-live plan and sign-off document

## When To Use It

- When all integrations are validated and ready
- Final step before go-live

## Inputs

- Validation reports for all integrations
- Implementation completion status
- User training completion
- Go/no-go criteria

## Outputs

- Go/no-go assessment document
- Detailed cutover plan
- Rollback procedures
- Go-live sign-off
