# Shopify PIM Selection Guide

**Entry Point**: `dept-shopify-pim-selection`

Structured decision guide for selecting appropriate product information management (PIM) platform for Shopify ecosystem. Uses quantitative complexity scoring and side-by-side evaluation of leading platforms (Akeneo, Bluestone, Inriver) plus assessment of staying without dedicated PIM.

## What This Skill Does

- Gathers product data requirements and complexity factors
- Calculates PIM complexity score using weighted criteria
- Evaluates Akeneo, Bluestone, Inriver, and skip-PIM approaches
- Assesses Shopify integration requirements and sync patterns
- Produces PIM selection recommendation with implementation timeline and cost analysis

## When To Use It

- Building new Shopify ecosystem and need PIM decision
- Current product information scattered across multiple systems
- Need to centralize product data governance
- Evaluating whether dedicated PIM is necessary vs. keeping Shopify as source of truth

## Inputs

- Product portfolio data (SKU count, variant depth, channels, enrichment needs)
- Data governance requirements
- Budget constraints
- Timeline expectations

## Outputs

- PIM complexity score (0-100)
- Platform evaluation comparison (Akeneo vs Bluestone vs Inriver vs No-PIM)
- Selection recommendation with clear rationale
- Implementation timeline (3-12 months typical)
- Cost analysis including licensing, integration, and operational costs
