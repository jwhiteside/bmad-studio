# Shopify Integration Build

**Entry Point**: `dept-shopify-integration-build`

Build and configure specific platform integrations. Install and configure platforms, set up APIs and webhooks, implement data mapping, test integrations, and document configuration for operations.

## What This Skill Does

- Configures platform setup (app installation, credentials, access)
- Sets up integration connections (APIs, webhooks, authentication)
- Defines and implements data mapping (attribute mapping, transformations)
- Tests integration with sample data, error scenarios, edge cases
- Documents integration configuration and operational runbook

## When To Use It

- During implementation phase when building individual integrations
- For each integration defined in architecture plan

## Inputs

- Integration specification from architecture
- Platform access and credentials
- Data mapping requirements
- Test data and scenarios

## Outputs

- Working integration between platforms
- Data mapping specification
- Integration testing results
- Operational runbook
