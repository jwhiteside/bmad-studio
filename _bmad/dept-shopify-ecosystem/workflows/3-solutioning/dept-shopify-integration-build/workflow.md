# Workflow: Shopify Integration Build

## Overview

Build and configure specific platform integrations. Set up APIs, webhooks, data mapping, and testing for each integration.

## Prerequisites

- Both platforms implemented and accessible
- Integration specification documented
- Platform access credentials available
- Test data prepared

## Steps

1. **Integration Kickoff**: [step-01-init](steps/step-01-init.md)
2. **Platform Configuration**: [step-02-configure](steps/step-02-configure.md)
3. **Data Mapping**: [step-03-data-mapping](steps/step-03-data-mapping.md)
4. **Integration Testing**: [step-04-test](steps/step-04-test.md)
5. **Documentation**: [step-05-document](steps/step-05-document.md)

## Completion Criteria

- Integration configured and working
- Data mapping validated with sample data
- All tests passing
- Runbook documented
- QA sign-off
