# Workflow: Shopify Data Flow Design

## Overview

Design detailed data flows for specific integration scenarios with field mapping, transformations, and error handling.

## Prerequisites

- Integration scenario clearly defined
- Source and destination data models documented
- Business transformation rules understood
- Error handling requirements known

## Steps

1. **Data Flow Kickoff**: [step-01-init](steps/step-01-init.md)
2. **Source Analysis**: [step-02-source-analysis](steps/step-02-source-analysis.md)
3. **Mapping Design**: [step-03-mapping](steps/step-03-mapping.md)
4. **Error Handling Design**: [step-04-error-handling](steps/step-04-error-handling.md)
5. **Specification Document**: [step-05-document](steps/step-05-document.md)

## Completion Criteria

- Data flow thoroughly analyzed
- Field mapping complete and validated
- Transformation rules documented
- Error handling strategy designed
- Specification document complete
