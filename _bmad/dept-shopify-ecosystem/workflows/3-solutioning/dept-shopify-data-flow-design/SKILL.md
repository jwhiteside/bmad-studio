# Shopify Data Flow Design

**Entry Point**: `dept-shopify-data-flow-design`

Design detailed data flows for specific integration scenarios. Analyze source data, design field mappings, transformation rules, error handling, and reconciliation patterns.

## What This Skill Does

- Analyzes source system data structure and events
- Designs field-level mapping and transformation rules
- Designs error handling, retry, and reconciliation patterns
- Produces data flow specification document
- Validates design with stakeholders

## When To Use It

- During design phase for complex data flows
- When designing product sync, customer events, order lifecycle, etc.

## Inputs

- Integration scenario description
- Source and destination data models
- Business transformation rules
- Error handling requirements

## Outputs

- Data flow specification document
- Field-level mapping table
- Transformation rule definitions
- Error handling design
## Workflow

1. **Data Flow Analysis**: Identify and understand source data
2. **Mapping Design**: Design field mapping and transformations
3. **Error Handling**: Design error handling and recovery
4. **Documentation**: Produce specification document
