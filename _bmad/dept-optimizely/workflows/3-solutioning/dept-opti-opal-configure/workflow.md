# dept-opti-opal-configure Workflow

**Workflow Name:** Configure Optimizely Opal AI Agents
**Phase:** Phase 3 — Solutioning
**Duration:** 3–5 days

## Workflow Overview

This workflow guides you through designing and deploying Optimizely Opal AI agents to augment editorial workflows. Opal AI provides capabilities for content generation, translation, personalization, and workflow automation.

## Key Principle

**AI Augments, Not Replaces.** Agents support editorial team workflows; content specialists retain approval authority and editorial judgment.

## Workflow Structure

```
step-01-init
    ↓ (use case assessment)
step-02-agent-design
    ↓ (agent architectures defined)
step-03-evaluate-deploy
    ↓ (agents tested and deployed)
DECISION GATE → [Approval / Preferred / Caution]
```

## Step Overview

- **Step 1:** Identify AI use cases (content generation, translation, SEO, personalisation), assess current workflows, document pain points
- **Step 2:** Design agent architectures, configure tools (system, CMS connectors, custom), define instructions, set up RAG knowledge sources
- **Step 3:** Design evaluations, test agents, deploy to Agent Directory, configure credit tracking

## Success Metrics

- Use cases identified and prioritized (5+)
- Agents designed and documented (3+ agents)
- Evaluation pass rate >= 80%
- Agents deployed to Agent Directory
- Editorial team trained on agent usage
- Credit usage < allocated budget

## Next Step

After approval, proceed to Phase 4 implementation and deployment.
